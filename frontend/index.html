About The Project

The Challenge

For a robot to interact intelligently with the world, it needs to understand not just what an object is, but precisely where it is and how it's oriented. This project tackles this core robotics challenge by building a system to predict an object's 6D Pose (3D position + 3D rotation) from a standard 2D image.

The Dataset

The model was trained for **20 epochs** on over 50,000 images from the YCB-Video dataset, a standard benchmark in robotics research. This dataset features 21 common household objects (e.g., cracker box, mustard bottle, banana, clamps) in various cluttered scenes, providing a realistic and challenging training environment.

The Model

The solution is a custom `PoseNet` model built in PyTorch. It uses a pre-trained ResNet50 backbone for powerful feature extraction, with a custom regression head to predict the final pose. The model's high precision is demonstrated by its very low final validation error.
